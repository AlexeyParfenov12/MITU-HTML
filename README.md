# MITU-HTML
## Запуск проекта

Эта инструкция поможет вам клонировать и запустить проект локально.

### Предварительные требования

Перед началом убедитесь, что у вас установлены:

•	Git — для клонирования репозитория и управления версиями.

•	Веб браузер — проект является HTML сайтом, поэтому для его просмотра достаточно браузера.


## Шаги по запуску

### Шаг 1. Клонирование репозитория
1.	Откройте терминал (командную строку) на вашем компьютере.
   
3.	Перейдите в папку, где хотите разместить проект:
   
```bash cd путь/к/папке```

5.	Выполните команду клонирования:
- Для HTTPS:
  
        bash git clone https://github.com/AlexeyParfenov12/MITU-HTML.git
- Для SSH (если настроен SSH ключ):
  
        bash git clone git@github.com:AlexeyParfenov12/MITU-HTML.git


4.	Перейдите в папку проекта:
   
```bash cd MITU-HTML```


### Шаг 2. Открытие проекта в браузере
Поскольку проект состоит из статических HTML файлов, установка дополнительных зависимостей не требуется.

1.	Найдите в папке проекта главный файл — обычно это ```index.html```.
   
3.	Откройте этот файл в любом веб браузере:
- Перетащите файл ```index.html``` в окно браузера.
- Либо кликните по файлу правой кнопкой мыши → «Открыть с помощью» → выберите браузер (Chrome, Firefox, Edge и т. д.).

  
### Шаг 3. Проверка работы
Убедитесь, что страница отображается корректно:
- Все элементы интерфейса (меню, изображения, блоки) должны быть видны.
- Ссылки должны работать и вести на нужные разделы или страницы.
- Проверьте адаптивность — как сайт выглядит на разных размерах окна браузера.

## Устранение неполадок

### Если возникли ошибки:
- «command not found» (например, git: command not found) — убедитесь, что Git установлен и добавлен в PATH. Скачайте его с официального сайта:
  ```git-scm.com```.
- Ошибка доступа (при клонировании по SSH) — настройте SSH ключ для GitHub. Для быстрого старта используйте HTTPS ссылку.
- Файл не открывается в браузере — проверьте, что вы открыли именно ```index.html```, а не какую либо другую папку или файл. Убедитесь, что файл не повреждён.
- Изображения или стили не загружаются — убедитесь, что все файлы были успешно клонированы. Проверьте структуру папок: файлы css/, images/ и т. п. должны находиться рядом с index.html.

  
### Полезные команды Git
- Обновить проект до последней версии (если будут обновления):
          ```bash git pull origin main```
- Посмотреть статус файлов (изменённые, новые):
          ```bash git status```
